# 📖 About

This repository contains Cisco Packet Tracer (.pkt) lab files created while learning **CCNA** and Computer Networks.

The labs cover router configuration, switching, routing protocols, NAT, ACLs, SNMP, NTP, EtherChannel, QoS, IP addressing, and other essential networking concepts.

These practical exercises are useful for:

- CCNA Students
- Diploma & Engineering Students
- Networking Beginners
- Interview Preparation
- Lab Practice

# ▶️ Getting Started

Clone the repository:

```bash
git clone https://github.com/sarkalekartik/Cisco-Packet-Tracer-CCNA-Labs.git
```

Open the project:

```bash
cd Cisco-Packet-Tracer-CCNA-Labs
```

Launch any `.pkt` file using Cisco Packet Tracer.
